Formatação css

A formatação de CSS (Cascading Style Sheets) refere-se ao uso de regras e propriedades para
definir como os elementos HTML devem ser apresentados visualmente em uma página da web.
O CSS permite controlar a aparência de aspectos como cores, fontes, tamanhos, espaçamentos, layouts e muito mais.
O css permite aplicar estilos a diferentes elementos de maneiras variadas.
